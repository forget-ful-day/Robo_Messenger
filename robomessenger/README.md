# Робо Мессенджер

Современный веб-мессенджер с поддержкой реального времени, созданный для работы в GitHub Codespaces.

## Функции

- Регистрация и вход пользователей
- Обмен сообщениями в реальном времени
- Список контактов с поиском
- Чат с роботом-помощником (искусственный интеллект)
- Адаптивный дизайн для всех устройств
- Темная тема с градиентами

## Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Node.js, Express.js
- **Real-time**: Socket.IO
- **Стили**: CSS с использованием Flexbox и CSS Grid
- **Иконки**: Font Awesome

2. После загрузки Codespace, откройте терминал
